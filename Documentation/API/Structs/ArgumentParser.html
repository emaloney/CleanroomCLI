ArgumentParser

public struct ArgumentParser

Used to parse command-line arguments, optionally enforcing one or more explicit ArgumentDeclarations.

  • init(declarations:shortFormPrefix:longFormPrefix:)

    Constructs an ArgumentParser with the given argument declarations.

    Throws

    If one or more ArgumentDeclaration contains invalid or conflicting values, or if shortFormPrefix is not a shorter string than longFormPrefix.

    Declaration

    Swift

    public init(declarations: [ArgumentDeclaration], shortFormPrefix: String = "-", longFormPrefix: String = "--")
        throws

    Parameters

    declarations

    An array of declarations representing the arguments that may be be encountered on the command line.
    shortFormPrefix

    A string prefix used on the command line to signal a short-form argument. Defaults to “-”.
    longFormPrefix

    A string prefix used on the command line to signal a long-form argument. This string must be longer than shortFormPrefix. Defaults to “--”.
